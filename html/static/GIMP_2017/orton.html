Einen Orton-Effekt erzeugen

Der Orton-Effekt wird durch die Überlagerung zweier überbelichteter Bilder erzielt, von denen eines weichgezeichnet wurde.
In GIMP kann über das Ebenen-Werkzeug ein Duplikat erstellt werden und dann über den Gaußschen Weichzeichner unscharf gemacht werden.
Abb. 1:  Das Ausgangsbild für den Orton-Effekt

Zunächst wird über die Ebenen-Verwaltung (Strg + L) ein Duplikat der Ebene erzeugt. Dazu befindet sich eine Schaltfläche im unteren Teil des Werkzeugs. Der Modus der oberen Ebene wird anschließend auf Multiplikation gesetzt.
Abb. 2: Das Ebenen-Werkzeug

Der erste Zwischenschritt. Das Bild ist jetzt dunkler.
Abb. 3: Der Zwischenschritt

Jetzt muss die obere Ebene über den Gaußschen Weichzeichner unscharf gemacht werden. Der Gaußsche Weichzeichner ist unter Filter → Weichzeichnen → Gaußscher Weichzeichner zu finden.
Abb. 4: Der Gaußsche Weichzeichner

Je nach Weichzeichenradius kann das Ergebnis unterschiedlich aussehen.
